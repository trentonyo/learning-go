# Set up

## Local

<details>
<summary>Windows</summary>

### TL;DR

```bash
scoop install go
scoop bucket add extra
scoop install goland go
```

### Install GO SDK

```bash
scoop install go
```

### Setting up GOROOT

This is the location of your GO install, e.g. `~\scoop\apps\go\current`

Set this at `Settings > GO > GOROOT > (+) > Local`

</details>

<details>
<summary>OSX</summary>

### TL;DR

```bash
brew install go
brew install --cask goland
```

### Install GO SDK

```bash
brew install go
```

### Setting up GOROOT

This is the location of your GO install, e.g. `~/opt/homebrew/Cellar/go/1.22.3/libexec`

Set this at `Settings > GO > GOROOT > (+) > Local`

</details>
